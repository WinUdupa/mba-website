# MBA Conference Website - ICSAR 2026

Conference website for ICSAR 2026 (International Conference on Strategic Agility & Resilience), hosted by the Department of MBA at BNM Institute of Technology, Bengaluru.

## Getting Started

Install dependencies:
```bash
npm install
```

Start development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

## Conference Details

- **Dates:** April 16-17, 2026
- **Venue:** BNM Institute of Technology, Bengaluru, Karnataka, India
- **Theme:** Best Practices in Technology and Management for a Digital Era
